Agradecendo pelas maravilhosas aulas dadas até o momento.

Para iniciar os estudos gits foi criada uma pasta para organizar arquivos.
``` mkdir nome-pasta ```
Após entrar na pasta ``` cd nome-pasta ``` foi realizado o comando ``` git init ```, comando responsável para o rastreamento de arquivos do git, no qual é criado um diretório oculto ``` .git ```.

Após realizado os passos ditos acima, foi feito diversos arquivos onde foi aprendido a subir os mesmos atraves dos 3 passos, onde eles são: 
  1º: os untracked, vistos com ``` git status ``` ;
  2º: os arquivos adicionados no qual estão sendo encaminhados para um commit, com comando  ``` git add nome-aquivo.extensão ```;
  3º: arquivos commitados, atraves do comando  ``` git commit -m "mensagem desejada  ```;
  
Criamos também branches através do comando  ``` git checkout -b nome-do-branch ```, logo em seguida vimos  ``` Merge e Rebase  ``` no qual são responsáveis por enviar arquivos de um branch para outro.

Após todos passos adicionamos o repositório remoto em nosso repositório local para realizar a sincronização, através do comando  ``` git remote add origin https://github.com/neondotta/aulagit.git ```.

Feita a sincronização, utilizamos o  ``` push e pull ```, sendo push o responsável do envio dos arquivos ao servidor e pull responsável pela verificação da atualização dos branchs.

Por ultimos pra concluir vimos o  ``` git clone  ```  no qual clona os repositórios do servidor para o local e as  ```TAGS``` onde aprendemos a criar nossas versões e ver a funcionalidade de cada numero em uma versão EX: 0.1.1.
